# YtYt Website

Современный адаптивный веб-сайт, созданный по макету Figma с использованием Tailwind CSS.

## About Project

Этот проект разработан с использованием современных frontend технологий и подхода Mobile First.

Сайт включает:
- адаптивный дизайн
- современный UI
- чистую структуру
- responsive layout
- оптимизацию под разные устройства

## Built With
